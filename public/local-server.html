Serveur Local pour Impression Epson

Instructions :

  1. Téléchargez et installez Python sur votre ordinateur
  2. Ouvrez un terminal dans le dossier du projet
  3. Exécutez : python -m http.server 8080
  4. Votre serveur local sera accessible à : http://192.168.1.XXX:8080

Configuration imprimante :

URL à utiliser : http://[VOTRE_IP_LOCALE]:8080/api/print-order/escpos.txt

Test :

Testez d'abord : escpos.txt

Problèmes possibles :

  • L'imprimante ne peut pas accéder à HTTPS
  • L'imprimante ne peut pas accéder aux domaines externes
  • Firewall bloque les connexions sortantes
  • L'imprimante a besoin d'un serveur local
